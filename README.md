# Aluminum Falcon Controller
An electronics controller for the Aluminum Falcon (https://whats-an-aluminum-falcon.blogspot.com/). This controller allows you to play music to a bluetooth speaker from an SD card based on toggle switch input, honk a horn, and send signals to the vehicle brake lights.
